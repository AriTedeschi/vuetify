## Para instalar o projeto rode:
```
npm install
```

### Para rodar e compilar o projeto para desenvolver
```
npm run serve
```

### Para distribuir o projeto
```
npm run build
```

### Ajustar arquivos
```
npm run lint
```

### Para modificar as configurações
Veja [documentação](https://cli.vuejs.org/config/).
